Computação Gráfica

A Computação Gráfica é a área da Ciência da Computação que estuda a geração de imagens de cenas virtuais usando computadores. De maneira geral, a cena virtual é tridimensional, ou seja, é definida por três dimensões representadas pelos eixos X, Y e Z, semelhante a funções matemáticas. Existem cenas bidimensionais, onde o eixo da profundidade (Z) não é utilizado. A cena virtual possui uma câmera que simboliza o observador que visualiza a cena. Objetos tridimensionais também podem compor a cena, como cubos, esferas, prédios, carros, pessoas, etc. A partir da cena virtual 3D ou 2D, vários algoritmos são realizados para gerar a imagem 2D observada pela câmera, que será exibida na tela do dispositivo eletrônico.

Muitos cálculos matemáticos são envolvidos para a geração da imagem a partir da cena virtual e aplicação dos vários efeitos visuais existentes. De acordo com a quantidade e os efeitos empregados, muito poder de processamento as vezes é necessário para geração da imagem da cena. Por isso, um hardware específico para tarefas gráficas foi criado, as placas gráficas (graphics cards) com vários núcleos de processadores, muita memória e barramento interno de alta velocidade são hardwares próprios para tarefas de geração de imagens. O melhoramento do hardware ao longo dos anos contribuiu muito para o avanço e melhoria no resultado das imagens produzidas.

Este conteúdo, apesar de ser importante para o desenvolvimento da tecnologia, não me interessou, pois envolve uma área muito específica, na qual não é acessível a todos, e no caso, não tem uma prioridade no meio do conhecimento básico sobre a computação.

Engenharia de Software

O desenvolvimento de software de médio e grande porte é muito complexa, pois é necessário o gerenciamento dos recursos envolvidos no projeto como equipe e código-fonte, mantendo os prazos, custos e a qualidade. O processo de desenvolvimento de software é pela área da Engenharia de Software.

A Engenharia de Software é o ramo da ciência da computação que estuda princípios para guiar o desenvolvimento de grandes e complexos sistemas de software.

Várias questões importantes no desenvolvimento de software são estudadas na Engenharia de Software. A estimativa do custo financeiro do desenvolvimento do software é uma questão importante, pois é importante conhecer quanto custará para decidir por exemplo, se o software será criado ou não. A divisão do projeto do software em partes menores, emprega a ideia de dividir para conquistar, ou seja, reduzir o grande e complexo sistema em partes menores e mais simples. Assim é importante garantir que as partes menores serão compatíveis e integráveis. Ao desenvolver as partes menores e integrá-las, o sistema todo estará completo.
